# GUI 表单填写

## 何时使用
当需要在 GUI 界面中填写和提交表单时。

## 执行步骤

1. 截图并 OCR 识别当前界面，列出所有可见的表单字段
2. 对每个字段：
   a. 找到字段标签位置
   b. 点击其对应的输入区域
   c. 清空已有内容（Ctrl+A）
   d. 输入对应值
   e. 用 Tab 移到下一个字段
3. 下拉选择框：点击 → 找到选项文字 → 点击选项
4. 复选框/单选框：直接点击
5. 所有字段填写完成后，截图验证
6. 找到提交按钮并点击
7. 等待提交结果，截图验证是否成功

## 注意事项

- 填写前先了解所有必填字段
- 日期字段注意格式（yyyy-mm-dd 或 mm/dd/yyyy）
- 长文本字段可能需要滚动
- 提交后注意检查是否有验证错误提示
- 如果缺少必要信息，先询问用户
