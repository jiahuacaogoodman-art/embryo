# GUI 系统登录

## 何时使用
当需要通过桌面 GUI 界面登录一个系统时（没有 API 可用）。

## 执行步骤

1. 截图观察当前界面状态
2. 用 OCR 识别界面文字，确认是否在登录页面
3. 查找"用户名"或"账号"文字位置
4. 在其对应的输入框中点击并输入用户名
5. 查找"密码"文字位置
6. 在其对应的输入框中点击并输入密码
7. 查找"登录"或"提交"按钮并点击
8. 等待 2-3 秒，再次截图
9. OCR 验证是否登录成功（检查是否出现首页元素或错误提示）

## 注意事项

- 输入前先点击输入框确保获得焦点
- 输入密码前先清空可能存在的内容（Ctrl+A 全选后输入）
- 如果出现验证码，请求人工协助
- 登录失败时读取错误提示文字，判断是账号错误还是密码错误
- 不要连续失败超过 3 次，避免账号被锁定

## 常见问题

| 问题 | 解决方案 |
|------|---------|
| 输入框没有获得焦点 | 多点击一次，或用 Tab 键切换 |
| 页面还在加载 | 等待 3-5 秒后重新截图 |
| 弹窗遮挡 | 先关闭弹窗（找关闭按钮或按 Esc） |
| 验证码 | 请求人工协助 |
